# Installation Instructions
1. Install dependencies
``` 
npm install
```
2. Create db
```
createdb DATABASE_NAME -U USERNAME
```
3. Seed data
```
psql -d DATABASE_NAME -U USERNAME -f tables.sql
psql -d DATABASE_NAME -U USERNAME -f seed.sql
```
